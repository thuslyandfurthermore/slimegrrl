how to make a page for slimegrrl.life

  • make a folder in the github repository like /other/yourpage/
  • copy this file to /other/yourpage/index.html
  • put your stuff here in the "main" div
    • make sure to add a title in the <head>!
    • to color text like this add class="yourname"
  • add an entry to /include/footscroll.html to link to your new page
